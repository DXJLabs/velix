# VEIL Documentation

VEIL documentation is organized product-first.

Read the product and guide sections before technical references. The repository should explain why VEIL exists and what users can do before exposing SDK, transport, ABI, cryptography, contract, or source-analysis material.

## Strategy

VEIL is a private communication, negotiation, payment memo, and escrow platform built on Starknet Privacy Pool.

The SDK, contracts, transports, encryption, helper contract, indexer, and research notes are supporting implementation layers. They are kept under technical documentation and should not be the first thing a new visitor sees.
