# Activity

Activity gives users a cross-channel view of pending actions and history.

## User Flow

1. Open Activity.
2. Review pending actions.
3. Select an action such as release escrow, send payment, or sign memo.
4. Review recent channel history.
5. Return to the relevant Deal Room when more context is needed.

## User Expectations

- Pending actions should be clear and actionable.
- History should show what happened and which channel it belongs to.
- Activity should help users avoid missing important deal steps.
