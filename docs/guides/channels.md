# Channels

Channels are the primary workspace in VEIL.

Each channel represents a private deal context between participants.

## User Flow

1. Open the channel list.
2. Search or select a channel.
3. Review the active counterparty, status, and next action.
4. Open the Deal Room.
5. Continue messaging, negotiation, payment memo, or escrow work.

## User Expectations

- Channels should make deal status scannable.
- The active channel should show the next useful action.
- Users should not lose context when moving between message, payment, and escrow views.
