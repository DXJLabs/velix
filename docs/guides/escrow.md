# Escrow

Escrow helps two parties coordinate a deal before settlement is completed.

VEIL presents escrow as a product workflow: create, confirm, activate, release, settle, or cancel.

## User Flow

1. Open the Escrow view from a Deal Room.
2. Review buyer, seller, asset, payment reference, and current status.
3. Confirm deposit milestones when ready.
4. Activate escrow after required confirmations.
5. Release or settle when the deal is complete.
6. Cancel if the deal should not proceed and cancellation is still available.

## User Expectations

- Escrow status should be clear at every step.
- Users should know which party needs to act next.
- Escrow actions should be connected to messages, offers, and payment memos in the same channel.
