# Settings

Settings control privacy defaults, notifications, wallet controls, sessions, and local account behavior.

## User Flow

1. Open Settings.
2. Choose the default privacy mode.
3. Decide whether to require confirmation before Unshield.
4. Decide whether to require a memo before payment.
5. Manage notifications.
6. Review security and session controls.
7. Manage wallet connection or local cache.

## User Expectations

- Privacy choices should be understandable.
- Riskier actions should require explicit confirmation.
- Session and wallet controls should be easy to find.
