# Wallet

The wallet connects the user to Starknet actions inside VEIL.

## User Flow

1. Open Wallet or click Connect Wallet.
2. Complete the wallet connection flow.
3. Review account address and network status.
4. Check asset readiness.
5. Return to channels or activity.

## User Expectations

- Wallet status should be visible before payment or escrow actions.
- The selected network should be clear.
- Users should be able to copy their address, refresh balances, switch wallet, or disconnect.
