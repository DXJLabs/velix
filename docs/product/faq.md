# FAQ

## Is VEIL an SDK?

No. VEIL is a product. The SDK supports the product but is not the repository entry point.

## What is a Deal Room?

A Deal Room is a private channel where users keep messages, payment memos, offers, proof references, and escrow status together.

## What is Shield mode?

Shield mode is the user-facing privacy mode for actions that should use Privacy Pool when the environment supports it.

## What is Unshield mode?

Unshield mode is an explicit direct/public path. It is useful for testing and flows where privacy is not required.

## Does VEIL replace Privacy Pool?

No. VEIL is the product layer built around Privacy Pool. Privacy Pool is the privacy foundation for shielded flows.

## Does VEIL custody funds today?

The current escrow contract records workflow state. Current documentation should not describe it as token custody or automatic asset release.

## Where should developers start?

Developers should read the product and architecture sections first, then use [Technical Documentation](../technical/README.md) as a reference.
