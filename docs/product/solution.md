# Solution

VEIL provides a private Deal Room for Starknet payments and escrow coordination.

Each channel keeps messages, memos, offers, proofs, payment context, and escrow milestones together. Users can choose Shield mode for privacy-preserving flows when available, or Unshield mode when they intentionally want a direct public/testnet path.

## Product Capabilities

- Private deal channels.
- Encrypted onchain messaging.
- Payment memo preparation and signing.
- Offer and counter-offer negotiation.
- Escrow status tracking.
- Proof reference attachment.
- Wallet readiness and activity review.
- Clear Shield and Unshield choices.

## STRK20 Privacy Pool Alignment

The RFP-aligned product value is:

- encrypted onchain messaging,
- payment memos,
- escrow negotiation,
- metadata-resistant communication.

The SDK, helper contract, cryptography, transports, indexer, and Privacy Pool integration are implementation details that support these user outcomes.
