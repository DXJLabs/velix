# User Journey

## 1. Connect Wallet

The user connects a Starknet wallet and checks account readiness.

## 2. Open A Channel

The user creates or opens a private Deal Room with a counterparty.

## 3. Discuss Terms

The parties exchange private messages and share deal context.

## 4. Negotiate

One party creates an offer. The other can accept, reject, or counter.

## 5. Add A Payment Memo

Before payment, the user adds a memo that explains what the payment represents.

## 6. Coordinate Escrow

The parties create escrow, confirm deposits, activate the deal, and release or cancel according to the outcome.

## 7. Review Activity

The user reviews pending actions and channel history.

## 8. Adjust Privacy

The user chooses Shield or Unshield behavior and manages privacy defaults in settings.
